# IPL Data Visualization Project 🏏

This project contains comprehensive data analysis and interactive visualizations of IPL (Indian Premier League) cricket data.

## 📊 Project Overview

This Jupyter notebook analyzes IPL cricket data to provide insights into team performance, player statistics, and match trends through various data visualization techniques.

## 📁 Files

- **Main Analysis**: `CSIT553_Project 1_Tabular Data Visualization_IPL_Sai Rohini Godavarthi & Rahul Ega.ipynb` - Complete data analysis and visualization notebook
- **Data Files**:
  - `deliveries.csv` - Ball-by-ball delivery data (18+ MB)
  - `matches.csv` - Match-level information and results
  - `most_runs_average_strikerate.csv` - Player performance statistics
- **Presentation**: `CSIT553_Project 1_IPL_PPT_Sai Rohini Godavarthi & Rahul Ega.pptx` - Project presentation
